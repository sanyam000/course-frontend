# 🎓 Course Management System

This is a full-stack application to manage courses and their instances, built using:

- 📦 **Backend**: Java Spring Boot
- 💻 **Frontend**: ReactJS
- 🛢️ **Database**: PostgreSQL
- 🐳 **Dockerized** with Docker Compose
